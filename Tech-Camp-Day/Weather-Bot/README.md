# Weather Bot

LINE Bot รายงานสภาพอากาศและแจ้งเตือนฝนตก โดยดึงข้อมูลแม่นยำจาก OpenMeteo API

## ฟีเจอร์หลัก
- **เลือกจังหวัด**: พิมพ์ชื่อจังหวัดที่ต้องการติดตามสภาพอากาศ (เช่น "เชียงใหม่", "กรุงเทพ")
- **รายงานประจำวัน**: ส่งสรุปสภาพอากาศ อุณหภูมิสูงสุด/ต่ำสุด และโอกาสฝนตก ทุกวันเวลา 08:30 น.
- **ระบบเตือนภัย**: แจ้งเตือนล่วงหน้า 1 ชั่วโมง หากตรวจสอบพบว่าสภาพอากาศจะเลวร้าย (ฝนตกหนัก/พายุ) ในจังหวัดที่คุณเลือก

## วิธีใช้งาน
1. **เริ่มใช้งาน**: กด Follow บอท หรือพิมพ์ชื่อจังหวัดใหม่ที่ต้องการ
2. **เปลี่ยนจังหวัด**: พิมพ์ชื่อจังหวัดใหม่ได้ตลอดเวลา

## การตั้งค่า
- ข้อมูลจังหวัดและผู้ใช้จะถูกเก็บใน SQLite (`data/db.js`)
- การตั้งเวลาส่งข้อความใช้ `node-cron`
- ตรวจสอบให้แน่ใจว่า `.env` มี `LINE_CHANNEL_ACCESS_TOKEN` และ `LINE_CHANNEL_SECRET` ครบถ้วน
