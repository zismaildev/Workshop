# Vocab Flash Card Bot

LINE Bot ช่วยท่องศัพท์ภาษาอังกฤษ พัฒนาทักษะภาษาผ่านการทายคำศัพท์ง่ายๆ

## ฟีเจอร์หลัก
- **ขอศัพท์ใหม่**: พิมพ์ `ขอศัพท์` บอทจะส่งคำศัพท์ภาษาอังกฤษมาให้
- **ทายความหมาย**: พิมพ์คำแปลภาษาไทยตอบกลับไป
    - ถ้าถูก: บอทจะชมเชย
    - ถ้าผิด: บอทจะเฉลยความหมายที่ถูกต้อง
- **ศัพท์รายวัน**: (Optional) ระบบแจ้งเตือนศัพท์ใหม่ทุกวันเวลา 18:00 น.

## วิธีใช้งาน
1. พิมพ์ `ขอศัพท์`
2. บอทส่งคำว่า "Apple" (สมมติ)
3. พิมพ์ตอบว่า "แอปเปิ้ล"
4. บอทแจ้งผลว่าถูกต้องหรือไม่

## การติดตั้ง
1. ติดตั้ง Dependencies: `npm install`
2. ตั้งค่า `.env`
3. รันโปรเจกต์: `npm start`
