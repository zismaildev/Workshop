# Horoscope Bot

LINE Bot สายมูเตลู สำหรับดูดวงรายวัน สุ่มคำทำนายเพื่อความบันเทิงและเสริมกำลังใจ

## ฟีเจอร์หลัก
- **ดูดวง**: สุ่มคำทำนาย 4 ด้านหลัก (ความรัก, การงาน, การเงิน, สุขภาพ)

## วิธีใช้งาน
- พิมพ์คำว่า `ดูดวง` ลงในแชท บอทจะตอบกลับด้วยคำทำนายประจำวัน

## การติดตั้งและรันโปรเจกต์
1. ติดตั้ง Dependencies:
   ```bash
   npm install
   ```
2. ตั้งค่า Environment Variables ในไฟล์ `.env`:
   - `LINE_CHANNEL_SECRET`
   - `LINE_CHANNEL_ACCESS_TOKEN`
3. รันโปรเจกต์:
   ```bash
   npm start
   ```
